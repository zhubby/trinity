# Changelog

## 2026-05-18

### Added
- Hold-to-record dictation manager with Idle, Recording, Transcribing, and Error states.
- Default microphone recording through `cpal`.
- WAV encoding for recorded audio.
- ElevenLabs Speech to Text multipart client.
- Direct focused-input text injection through `enigo`.
- Unit coverage for transcription response parsing, WAV encoding, injection errors, and manager state transitions.
